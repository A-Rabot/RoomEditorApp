Description

With QuickVent you can get a head start on Ventilation calculations by exporting room names, room numbers and areas to excel. Alternatively, QuickVent can create ventilation calculation spreedsheets in excel and auto fill occupant airflow requirements for rooms that are named Office, Corridor, Storage or Conference.

General Usage Instructions

Hit the QuickVent Button to pull up the main menu.

Once in the main menu hit browse to locate your ventilation calculation excel spreedsheet. 

Then select the tab from the sheet pulldown menu.

From here you have the options to create a Rate Table Tab, export room names, room numbers and/or room areas.  

Next select the direction that you want the information to populate in excel.  

Afterwards hit select rooms to select the rooms that you'd like to export.  You have the option to select a single room or multiple rooms.  

After you hit ok, the main menu returns. Hit ok to finish.

Installation/Uninstallation

The installer that ran when you downloaded this plug-in from Autodesk App store has already installed the plug-in. You may need to restart the Autodesk product to activate the plug-in. To uninstall this plug-in, exit the Autodesk product if you are currently running it, simply rerun the installer by downloading it again from Autodesk App store, and select the 'Uninstall' button, or you can uninstall it from 'Control Panel\Programs\Programs and Features' (Windows 7/8/8.1/10) or Control Panel > Add or Remove Programs (Windows XP), just as you would uninstall any other application from your system.
